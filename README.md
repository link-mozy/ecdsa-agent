# ECDSA-AGENET

ECDSA Tutorial

`ECDSA 알고리즘`과 `tokio`를 사용한다.

## ecdsa-agent-server

테스트 실행 방법

```bash
$ cargo run
ecdsa-agent 0.0.1
tester

USAGE:
    ecdsa-agent [SUBCOMMAND]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    help    Prints this message or the help of the given subcommand(s)
    run     run ecdsa-agent
    stop    stop ecdsa-agent%  
```

```bash
## ecdsa-agent run
$ cargo run -- run

## ecdsa-agent stop
$ cargo run -- stop
```
